# Public Cam Dashboard

A world map of **publicly listed, I SWEAR ITS NOT ILLEGAL** webcams and traffic cameras. Uses feeds from:

TfL JamCams, Caltrans, Windy Webcams API, and Earthcam

All sources are operator-published feeds with public APIs or curated public listings, so this project doesn't
use any non-publically available private cams

You can also pull up all the current live Starlink locations (updated per 10-seconds-ish), and every military base
location for all countries across the globe with their respective names.
